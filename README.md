# K-Indicator (한국 국가지표 AI 자동화 분석 시스템)

이 프로젝트는 한국의 주요 국가 경제지표(기준금리, 소비자물가상승률, 무역수지, 수출입 증가율 등)가 발표될 때마다 공식 부처 및 한은 보도자료 원본을 자동으로 수집, 분석 및 시각화하는 로컬 자동화 툴입니다.

뉴스 기사를 크롤링하는 것이 아닌 **정부 부처(정책브리핑) 및 한국은행 공식 보도자료의 첨부 문서(PDF/HWPX)**를 직접 다운로드하여 텍스트를 추출한 뒤 AI 분석을 가동하므로 왜곡 없는 원본 통계를 실시간으로 분석합니다.

## 시스템 아키텍처

- **Backend**: Python, FastAPI, SQLite3
- **Scrapers**: 
  - **한국은행 (BOK)**: 공식 공보 AJAX 게시판 크롤러
  - **기재부, 산업부, 관세청, 통계청 (Korea.kr)**: 정부 정책브리핑 RSS 및 보도자료 게시판 크롤러
- **Parsers**: Native XML Parser (`zipfile`, `xml.etree.ElementTree`)로 HWPX, DOCX 파일 내 텍스트 추출, `pypdf`를 통한 PDF 텍스트 추출
- **AI Engine**: Google Gemini API (`gemini-1.5-flash`)를 활용한 핵심 요약, 시장 영향 평가 및 지표 수치 구조화 JSON 추출 (API 키 미설정 시 간이 Regex 룰베이스 추출 기능으로 자동 폴백)
- **Frontend**: Glassmorphism 디자인의 바닐라 CSS 테마 대시보드, Chart.js 활용 시계열 트렌드 시각화

## 실행 방법 (Windows)

1. 프로젝트 폴더의 `run.bat` 파일을 더블 클릭하여 실행합니다.
2. 로컬 웹 서버가 가동되며 자동으로 브라우저에 대시보드 페이지(`http://127.0.0.1:8000`)가 열립니다.
3. 대시보드 우측 상단 혹은 설정 탭의 **[지금 수집]** 또는 **[동기화 실행]** 버튼을 클릭하면 크롤링 및 분석 작업이 백그라운드에서 진행됩니다.

## 주요 기능 및 탭 설명

1. **종합 대시보드**:
   - 최신 기준금리, 물가상승률, 무역수지, 수출증가율 KPI 카드 표시
   - 기준금리 vs 소비자물가상승률 이중축 추이 차트 제공
   - 수출증가율 vs 무역수지 조합 차트 (수지 흑자/적자에 따른 막대 색상 가변 처리)
2. **보도자료 목록**:
   - 크롤링된 모든 공식 발표 보도자료 목록 조회
   - 기관별 필터 및 텍스트 실시간 필터링 기능
   - 각 보도자료 행을 클릭하면 모달 팝업이 활성화되어 **AI 핵심 요약**, **시장 영향 분석**, **추출 경제지표 목록**, **추출 원본 텍스트**, **로컬 첨부파일 바로가기**를 제공
3. **환경 설정**:
   - Gemini API Key를 등록하여 고성능 AI 문서 분석 기능을 활성화할 수 있습니다.
   - 전체 강제 동기화 크롤러 수동 가동
   - 로컬 파일 다운로드 폴더(`data/downloads/`) 탐색기 직접 열기 연동

## 환경 요구사항
- Python 3.10 이상
- 패키지 목록 (`requirements.txt` 참조):
  - `pypdf`, `fastapi`, `uvicorn`, `requests`, `beautifulsoup4`, `feedparser`, `google-generativeai`
